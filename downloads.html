Bootstrap

Originally created by a designer and a developer at Twitter, Bootstrap has become one of the most popular front-end frameworks and open source projects in the world. It allows you to make great looking Responsive Web sites really easily. It can be downloaded from https://getbootstrap.com/

it's an HTML, CSS and Javascript framework so essentially it's a collection of CSS and javascript files which enable you to create great looking Responsive Web sites responsive meaning they look completely different depending on the size of the user screen that's really important for web developers at the moment because the use of mobile phones is increasing very rapidly.

You can download the files by clicking on the download button and upload to your server and link to the style sheets in the head of your HTML file or if you you click on the get started button and scroll down the page you can copy the code for your index.html page using the starter template.

Bootstrap CDN

An easy way to install Bootstrap on a project is to use a Content Delivery Network (CDN). There are several benefits to using a CDN to host Bootstrap: Your website does not take a bandwidth hit for downloading the files because they are hosted somewhere else. Often CDN files are pre-cached because so many people are using them, so they help your pages load more quickly. You donâ€™t need command-line access to install Bootstrap. It can be installed more quickly on any page you manage with just a few lines in the HTML. If your adding a style sheet its recommended that you add it below the bootstrap CSS link in the starter template.
